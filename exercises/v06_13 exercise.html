Exercise - Count points and lives

Expand the CoinClicker mini game with counters

You are going to implement my Coinclicker-game, but in very small steps. In previous exercises you added animations and events. For this version you don't have to worry about animations, but purely on the UI.

Download the zip-file: coinclicker_score.zip and unpack it to your exercises folder.

If you really want to, you can work with your existing files - just add the ui_elements from the HTML, and the few lines of CSS that sets it up. You also need the broken-heart.svg file. There are no other changes to the basic game.

Add a script.js file, and use a console.log to verify that it runs.

Then create event listeners on all the game_elements, the coins, the bomb and the heart.

When you have the event listeners up and running, add variables to keep track of points and lives.

In this version, just console.log the variables when they change.

The rules for the game are as follows:

  • The player starts with 3 lives, and 0 points
  • When a coin is clicked, the player earns 1 point
  • When a bomb is clicked, the player loses a life
  • When a heart is clicked, the player gains a life

In this version, don't worry about getting a negative number of lives.

You are going to re-use all the results in a later exercise, so don't throw them away!
