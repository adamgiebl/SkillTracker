Exercise - Building functions

create the functions for the CoinClicker game

You are going to implement my Coinclicker-game, but in very small steps. In the last exercise you played a bit around with querySelector and classList, and got something moving on the screen. - In this exercise, you are going to create some of the functions for use in the game.

Most of the functions will use some of the code-lines you have already written, so continue working in the same project!

Create the following functions, one at a time. Everytime you have written a function, call it from the inspector/console, by going to your browser, and type the name of the function, followed by a set of parantheses ().

Include a console.log line in each function, so you get feedback that it runs, even if you can't see the result on the screen.

Create these functions:

  1. startBomb() that starts the bomb falling continuously.
  2. startCoins() that starts all three coins falling, with at least two different speeds.
  3. pauseCoin1() that pauses the falling of just the first coin
  4. coin2Clicked() that pauses the falling of coin2, and zooms it as if it was clicked.
  5. bombClicked() that pauses the falling of the bomb, and makes it explode
  6. explosion() that causes the screen to shake
  7. bombRestart() that removes the explosion from the bomb, and unpauses it

If you finish quickly, feel free to also add similar function for the other coins, and the heart.
You are going to re-use all the results in a later exercise, so don't throw them away!
