Exercise - Add and remove animations

add animations to the CoinClicker game

You are going to implement my Coinclicker-game, but in very small steps. In this exercise you just have to play a bit around with querySelector and classList - in the following exercises you are going to build something more useful.

Download the zip-file: coinclicker_mini.zip and unpack it to your exercises folder.

Add a script.js file, and use a console.log to verify that it runs.

Then use querySelector and classList to achieve the following results – you might want to comment out some results, to avoid conflicts.

You should only add classes with JavaScript - you can try to turn them off and on with the inspector .cls tool, though.

  1. Make the first coin - #coin_container1 fall continuously
  2. Make the second coin - #coin_container2 also fall continuously, but add a different speed to it
  3. Pause the first coin - use the inspector to "un-pause" it for a while, so it shows up, then pause it again with the inspector.
  4. Make the third coin (the sprite) zoom in, as if it was clicked
  5. Make the bomb "explode" - there's missing a keyframe animation for this - so you need to create one!
  6. Make the entire game "shake" as an effect of the explosion

You are going to re-use all the results in a later exercise, so don't throw them away!
