Exercise - Build the CoinClicker mini game

- just clicking, and changing animations

You are going to implement my Coinclicker-game, but in very small steps.
In the last exercise you built a lot of functions, and now it is time to add some structure, and chain them together, so the game can be "played".
At least the user should be able to click on coin2 and make it appear as if it went into the score - or click the bomb, and make it explode and shake the screen.
A fun game!

Use this State Machine Diagram for the structure of the game. You will notice that you already have most of the functions, so it is just a matter of adding and removing the correct event listeners.
When you are done, there should be no need to type anything in the console, the game should start and run by itself.

Note that you might want to check up on how to restart animations!

When you are done, you can follow a similar procedure for your own game - and remember, it is just about getting something moving and clicking.
